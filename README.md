# 🎓 Система автоматизации документооборота кафедры

Веб-приложение на **Streamlit** для автоматизации учебного процесса кафедры:
парсинг Excel-файлов нагрузки, хранение данных в SQLite, редактирование записей
и генерация индивидуальных планов преподавателей в формате `.xlsx` / `.docx`.

---

## 📋 Возможности

| Функция | Описание |
|---|---|
| 📂 **Импорт нагрузки** | Загрузка Excel-файлов распределения нагрузки кафедры |
| 📥 **Импорт индивидуальных планов** | Парсинг готовых `.xlsx`-планов преподавателей |
| ✏️ **Редактирование** | Правка ФИО, часов и дисциплин прямо в браузере |
| 📄 **Генерация планов** | Автозаполнение индивидуального плана по шаблону `.xlsx` |
| 📝 **Генерация документов** | Формирование отчётов по шаблонам `.docx` |
| 🗄️ **Локальная БД** | Хранение данных в двух SQLite-файлах (без сервера) |
| 🔍 **Поиск и фильтрация** | Поиск преподавателей, дисциплин, групп по базе данных |

## 🗂️ Структура проекта

```
new_prikol-main/
├── app.py                  # Основное Streamlit-приложение (веб-интерфейс)
├── parser_engine.py        # Парсер Excel-файлов нагрузки и индивидуальных планов
├── excel_filler.py         # Заполнение шаблона индивидуального плана (.xlsx)
├── local_db.py             # Локальная БД: эмуляция MongoDB API поверх SQLite
├── ai_parser_helper.py     # Вспомогательный модуль AI-парсинга
├── template_helper.py      # Работа с шаблонами .docx
├── db_setup.py             # Инициализация и очистка базы данных при запуске
├── queries.py              # Демонстрационные запросы к базе данных
├── requirements.txt        # Зависимости Python
├── run.bat                 # Скрипт быстрого запуска (Windows)
├── university_db.sqlite    # БД преподавателей и импортированных файлов
└── workload_db.sqlite      # БД нагрузки (сотрудники, группы, предметы, практики)
```

## 🚀 Быстрый запуск (Windows)

Просто дважды кликните на **`run.bat`** — скрипт автоматически:
1. Создаст виртуальное окружение `venv`
2. Установит все зависимости из `requirements.txt`
3. Инициализирует базу данных SQLite
4. Запустит веб-приложение

```
run.bat
```

После запуска откройте браузер по адресу: **http://localhost:8501**

## 🛠️ Ручная установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git
cd new_prikol-main

# 2. Создайте виртуальное окружение
python -m venv venv

# 3. Активируйте его
# Windows:
venv\Scripts\activate
# Linux / macOS:
source venv/bin/activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Инициализируйте базу данных
python db_setup.py

# 6. Запустите приложение
streamlit run app.py
```

Откройте браузер: **http://localhost:8501**

## 🗄️ База данных

Приложение использует **две независимые SQLite-базы**:

### `university_db.sqlite`
| Коллекция | Содержимое |
|---|---|
| `employees` | Преподаватели и их нагрузка |
| `imported_files` | История импортированных файлов |

### `workload_db.sqlite`
| Коллекция | Содержимое |
|---|---|
| `employees` | Данные о сотрудниках кафедры |
| `departments` | Кафедры |
| `institutes` | Институты |
| `groups` | Учебные группы |
| `students` | Студенты |
| `subjects` | Дисциплины |
| `practices` | Практики |
| `unassigned_loads` | Нераспределённая нагрузка |
| `individual_plans` | Индивидуальные планы преподавателей |
| `imported_files` | История импорта |

> **Примечание:** `local_db.py` реализует MongoDB-совместимый API (`MongoClient`, `Collection`, `Cursor`, `ObjectId`) поверх SQLite, что позволяет при необходимости заменить хранилище на реальный MongoDB без изменения кода приложения.

## 📖 Использование

### 1. Импорт файла нагрузки
1. Откройте приложение в браузере
2. В боковом меню выберите раздел импорта
3. Загрузите Excel-файл с распределением нагрузки кафедры (`.xlsx`)
4. Дождитесь завершения парсинга — данные сохранятся в базу автоматически

### 2. Просмотр и редактирование
- В разделе **«Преподаватели»** отображается список всех загруженных сотрудников
- Нажмите на преподавателя для просмотра его нагрузки
- Используйте кнопки редактирования для изменения данных

### 3. Генерация индивидуального плана
1. Выберите преподавателя из списка
2. Загрузите шаблон `.xlsx` или используйте шаблон по умолчанию
3. Нажмите **«Сформировать план»**
4. Скачайте готовый заполненный файл

### 4. Проверка базы данных (демо-запросы)
```bash
python queries.py
```
Выведет список сотрудников, нагрузку по предметам и статистику.

## ❓ Устранение неполадок

| Проблема | Решение |
|---|---|
| `streamlit: command not found` | Убедитесь, что вирт. окружение активировано и зависимости установлены |
| Ошибка при запуске `run.bat` | Убедитесь, что Python добавлен в переменную среды `PATH` |
| Пустая таблица преподавателей | Импортируйте Excel-файл нагрузки через интерфейс приложения |
| `ModuleNotFoundError` | Выполните `pip install -r requirements.txt` |
| Ошибка чтения `.xls` файла | Убедитесь, что установлен `xlrd >= 2.0.1` |
