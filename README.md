# 🎓 Система автоматизации документооборота кафедры

Streamlit-приложение для автоматизации учебного процесса: парсинг Excel-файлов нагрузки, хранение в базе данных SQLite, редактирование и генерация индивидуальных планов преподавателей.

## 🚀 Возможности

- 📂 **Импорт нагрузки** — загрузка Excel-файлов распределения нагрузки кафедры
- ✏️ **Редактирование** — правка часов и ФИО прямо в браузере
- 📄 **Генерация планов** — автозаполнение индивидуального плана по шаблону `.xlsx`
- 🗄️ **Локальная БД** — все данные хранятся в SQLite (без сервера)

## 📋 Требования

- Python 3.9+
- Зависимости из `requirements.txt`

## ⚙️ Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git
cd ВАШ_РЕПОЗИТОРИЙ

# 2. Создайте виртуальное окружение
python -m venv venv

# 3. Активируйте его (Windows)
venv\Scripts\activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Запустите приложение
streamlit run app.py
```

Или просто запустите `run.bat` (Windows).

## 📁 Структура проекта

```
├── app.py                  # Основное Streamlit-приложение
├── parser_engine.py        # Парсер Excel-файлов нагрузки
├── excel_filler.py         # Заполнение шаблона индивидуального плана
├── local_db.py             # Эмуляция MongoDB поверх SQLite
├── template_helper.py      # Вспомогательные функции для шаблонов
├── db_setup.py             # Инициализация базы данных
├── requirements.txt        # Зависимости Python
└── run.bat                 # Скрипт быстрого запуска (Windows)
```
